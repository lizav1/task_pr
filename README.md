
### Налаштування та запуск
Clone repository:

	`git clone https://github.com/lizav1/task_pr.git`

	 `cd task_pr/`
 
 
Create virual env:

  	`$ python3 -m venv venv`
  
 	`$ source venv/bin/activate`


Install libraries for project:

  	`$ pip install -r requirements.txt `


Run project:

  	`$ python manage.py runserver `


Open link in browser:

  	`http://127.0.0.1:8000/`
